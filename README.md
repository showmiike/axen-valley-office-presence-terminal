Axen Valley Office Presence Terminal
Axen Valley Office Presence Terminal holo ekti modern ebong user-friendly attendance management system, jeta office-er dainandin uposthiti track korar jonno toiri kora hoyeche. Eti prothomoto Bangladesh-based supplement e-commerce operation-er team management-er kaje niyoito.

🚀 Features
Real-time Attendance: Employee-ra khub shohojei tader ID ebong naam select kore attendance mark korte pare.

Automated Status Tracking: System-ti automatically 09:30 AM-er porer entry-gulo ke "Late" hishebe mark kore.

Personal Employee Dashboard: Prottek employee tader nijer performance, present ratio, ebong last 5-10 diner history dekhte pare.

Secure Admin Panel: Admin login chara sensitive options (Download, Clear Records) hidden thake.

Monthly Reporting: Ek click-e pura masher attendance report Excel (XLSX) format-e download kora jay.

Data Persistence: LocalStorage maintain korar fole browser refresh dileo data haray na.

🛠️ Tech Stack
Frontend: HTML5, CSS3 (Product Sans Font, Animate.css).

Logic: Vanilla JavaScript (ES6+).

Library: SheetJS (XLSX) for report generation.

📖 How to Use
Attendance: Employee ID ebong Name select korun. Proyojon hole manual entry time dite paren, na hole current time-i count hobe.

Dashboard: Employee Name select korle niche auto-animate hoye personal dashboard chole ashbe.

Admin Access: Personal
